#  Climate Change Prediction & Scenario Analysis using LSTM (Cloud Deployed)

##  Overview
This project uses a Long Short-Term Memory (LSTM) deep learning model to predict multiple climate variables and perform scenario-based climate analysis. The application is cloud deployed, allowing users to interactively test different climate scenarios and view predictions in real time.

##  Features
- Multi-output climate prediction using LSTM
- Time-series forecasting
- Scenario-based climate analysis
- Interactive Gradio web application
- Cloud deployment
- Data visualization
- Real-time predictions

## Dataset
The model is trained using historical climate data containing:
- Carbon Dioxide (CO₂)
- Methane (CH₄)
- Solar Radiation
- Sea Level
- Precipitation
- Temperature Anomaly

## Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Gradio
- Hugging Face Spaces

1. Data Collection
2. Data Preprocessing

## Model Predictions
The model predicts:
- Carbon Dioxide (CO₂)
- Methane (CH₄)
- Solar Radiation
- Sea Level
- Precipitation
- Temperature Anomaly

## Scenario Analysis
Users can modify climate variables to simulate different future environmental conditions and observe the predicted impact on climate.

## Cloud Deployment
The application is deployed using Gradio and can be accessed through a web browser without any local installation.
